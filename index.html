Aydin Mustafa - BSc (Hons) Computing

Link N' GO - A Social Travel website

A project which implemented a cloud-based networking laboratory system based around the open source OpenNebula platform. The system allows students to deploy virtual machines which are customised to meet learning objectives and are available from any location using a variety of internet-connected devices. As well as increasing the accessibility of network lab resources, the system potentially reduces the cost of implementing and operating a network lab by enabling low cost, low power single board computers to perform the same functions as more expensive, performance-oriented desktop PCs. The resulting system is more flexible, accessible, scalable, cost-effective, and environmentally friendly than a traditional lab installation.
